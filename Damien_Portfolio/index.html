Project : Spectral Daze

Summary

This project was a small prototype for a video game that I was working on within the Unity3D engine. I deployed tools for the designers on my team to be able to create things without knowing how to program. These tools included json based editor menus to be able to create items, characters, enemies, and dialogue without having to touch a line of code.

Demo Video

There werer also many gameplay systems that were implemented by me. I created systems for managing things like particles, lights, sounds and any other elements that would cosntantly being created and destroyed. I created a object pooling system from scratch to help ease performance spikes caused by unity deleting and creating items. I also deployed input systems and enemy AI systems that were able to be manipulated by changing values. Everything within the project was obsifcutated so you were able to test and use any system on its own without having to deploy the rest, making it extremely easy to update and work on them, and test to make sure they were working, without loading up everything else. This was a .NET project. Some of the tools I used were Rider, VS Studio, and Unity3D. Blender was used for the art.

var shuffle = function (array) {

    var currentIndex = array.length;
    var temporaryValue, randomIndex;

    // While there remain elements to shuffle...
    while (0 !== currentIndex) {
        // Pick a remaining element...
        randomIndex = Math.floor(Math.random() * currentIndex);
        currentIndex -= 1;

        // And swap it with the current element.
        temporaryValue = array[currentIndex];
        array[currentIndex] = array[randomIndex];
        array[randomIndex] = temporaryValue;
    }

    return array;

};
